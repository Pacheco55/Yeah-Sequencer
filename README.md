**Secuenciador musical profesional con síntesis avanzada y efectos de estudio en tiempo real**

*Una poderosa estación de trabajo musical desarrollada por PIXELBITS Studios que combina la simplicidad de uso con capacidades de síntesis profesional, permitiendo a músicos, productores y entusiastas crear secuencias musicales complejas sin necesidad de conocimientos técnicos avanzados.*

---

## **¿Por qué PIXELBITS YEAH SEQUENCER?**

En el mundo de la producción musical digital, **la creatividad no debe limitarse por la complejidad técnica**. YEAH SEQUENCER democratiza la creación musical ofreciendo **herramientas de nivel profesional** en una interfaz intuitiva que permite desde composiciones simples hasta experimentación sonora avanzada.

### **Revoluciona tu Proceso Creativo**

**Síntesis Musical Avanzada**
Motor de síntesis multimodal con formas de onda clásicas (senoidal, cuadrada, triangular, diente de sierra) y síntesis FM avanzada con control total sobre modulación e índice. Generación en tiempo real sin latencia perceptible y calidad de audio profesional a 22kHz.

**Escalas Musicales Completas**
Biblioteca integral de 12 escalas musicales incluyendo modos griegos (Dórica, Frigia, Lidia, Mixolidia, Eólica, Locria), escalas pentatónicas (mayor y menor), escalas de blues (mayor y menor) y escalas tradicionales (mayor y menor natural).

**Sistema de Efectos Profesional**
Reverb espacial con control de saturación ajustable, vibrato expresivo con intensidad variable, delay estéreo con múltiples repeticiones y temporización precisa, filtros digitales de alta calidad (paso bajo, paso alto, paso banda, rechazo de banda).

**Modos de Composición Flexibles**
Modo manual para composición nota por nota con control individual de duración, modo aleatorio inteligente que respeta la escala seleccionada, secuencias de hasta 64 notas para composiciones extendidas y reproducción en bucle infinito para patrones rítmicos.

## **Guía de Inicio Rápido**

### **Primera Ejecución**

**Paso 1: Descarga e Instalación**
Descarga el archivo ZIP desde la sección de releases, extrae el contenido en una carpeta de tu elección, asegúrate de tener permisos de lectura/escritura en la carpeta y ejecuta el archivo PIXELBITS_YEAH_SEQUENCER.exe.

**Paso 2: Verificación de Audio**
El software detectará automáticamente tu dispositivo de audio, verifica que el volumen del sistema esté activado, prueba la reproducción con los controles de volumen general y efectos.

**Paso 3: Primera Secuencia**
Selecciona un tono base (recomendado: C), elige una escala (recomendado: Mayor para empezar), configura 8 notas como punto de partida y presiona "Generar Secuencia" en modo Aleatorio.

### **Controles Básicos**

**Configuración Musical**

<img width="1254" height="176" alt="Image" src="https://github.com/user-attachments/assets/07a839f0-8627-4dc8-ac05-ca6fe82f6c20" />

**Tono Base**: Selecciona la nota fundamental de tu composición (C, C#, D, D#, E, F, F#, G, G#, A, A#, B).

**Escala**: Elige entre 12 escalas musicales diferentes. Cada escala determina qué notas están disponibles para crear armonías coherentes.

**Octava**: Controla el registro tonal desde octava 0 (muy grave) hasta octava 7 (muy agudo). Octava 4 es el rango medio estándar.

**Modo de Generación**
**Manual**: Crea secuencias nota por nota haciendo clic en los botones de notas. Cada nota se reproduce inmediatamente para feedback auditivo.

**Aleatorio**: Genera secuencias automáticamente usando solo las notas de la escala seleccionada, garantizando coherencia musical.

### **Síntesis y Efectos**

**Tipos de Síntesis**

<img width="801" height="247" alt="Image" src="https://github.com/user-attachments/assets/4d641d17-3959-4a48-a15d-52ef4155fc2e" />

**Senoidal**: Sonido puro y suave, ideal para melodías etéreas.

**Cuadrada**: Sonido electrónico clásico, perfecto para música chiptune.

**Triangular**: Tono cálido entre senoidal y cuadrada.

**Diente de Sierra**: Sonido brillante y rico en armónicos.

**FM (Modulación de Frecuencia)**: Síntesis avanzada para sonidos complejos y experimentales.

**Control de Efectos**

<img width="1373" height="82" alt="Image" src="https://github.com/user-attachments/assets/4ce1f5a3-8ae3-4291-aada-15ecc92e03e1" />

**Reverb**: Añade espacialidad y profundidad al sonido.

**Vibrato**: Modulación de pitch para expresividad musical.

**Delay**: Ecos rítmicos con control estéreo para amplitud sonora.

**Filtros**: Moldeado tonal preciso con diferentes tipos de filtro.

## **Manual de Usuario Avanzado**

### **Composición Manual Avanzada**

**Secuencias Complejas**
Crea secuencias de hasta 64 notas para composiciones extendidas, usa diferentes octavas dentro de la misma secuencia para rango dinámico, combina escalas cambiando configuración entre secuencias y experimenta con duraciones variables para cada nota.

**Control de Duración Individual**
En modo manual, cada nota agregada genera automáticamente un slider de duración individual. Rango de duración: 0.1 a 2.0 segundos por nota. Ajusta duraciones para crear ritmos complejos y fraseo musical expresivo.

**Gestión de Secuencias**
**Deshacer**: Elimina la última nota agregada sin afectar el resto.

**Limpiar**: Borra toda la secuencia para empezar de nuevo.

**Visualización**: Monitor en tiempo real de todas las notas en la secuencia actual.

### **Síntesis FM Profesional**

**Parámetros de Modulación**
**Modulador**: Frecuencia del oscilador modulador (50Hz - 2000Hz recomendado).

**Índice FM**: Intensidad de la modulación (0.1 - 10.0 para rangos desde sutil hasta extremo).

**Aplicaciones**: Sonidos de campanas, texturas metálicas, efectos espaciales y timbres sintéticos únicos.

**Consejos de Síntesis**
Índices bajos (0.5-2.0) para modulación sutil, índices medios (2.0-5.0) para timbres complejos, índices altos (5.0-10.0) para efectos extremos y moduladores altos para brillantez adicional.

### **Sistema de Filtros Avanzado**

**Tipos de Filtro**
**Paso Bajo**: Elimina frecuencias agudas, suaviza el sonido.

**Paso Alto**: Elimina frecuencias graves, clarifica el sonido.

**Paso Banda**: Permite solo un rango específico de frecuencias.

**Rechazo de Banda**: Elimina un rango específico de frecuencias.

**Parámetros de Control**
**Frecuencia de Corte**: 50Hz - 8000Hz (punto donde el filtro comienza a actuar).

**Factor Q**: 0.1 - 10.0 (control de resonancia y ancho de banda).

## **Características Técnicas Avanzadas**

### **Motor de Audio**

**Especificaciones Técnicas**
Frecuencia de muestreo: 22,050 Hz, resolución: 16-bit estéreo, latencia mínima para respuesta en tiempo real, procesamiento multithread para estabilidad

**Síntesis en Tiempo Real**
Generación procedural de formas de onda, cálculos trigonométricos de alta precisión, envolvente ADSR automática para naturalidad sonora y optimización de memoria para secuencias largas.

### **Algoritmos de Efectos**

**Reverb Digital**
Implementación de delay con retroalimentación, control de saturación para calidez tonal, procesamiento estéreo para espacialidad y optimización CPU para uso en tiempo real.

**Delay Estéreo**
Hasta 8 repeticiones configurables, temporización variable de 0.05 a 1.0 segundos, procesamiento independiente por canal y efecto ping-pong para amplitud estéreo.

**Filtros Digitales**
Implementación Butterworth de segundo orden, respuesta en frecuencia lineal, factor Q variable para resonancia controlada y procesamiento de baja latencia.

## **Resolución de Problemas**

### **Problemas de Audio**

**Sin sonido al reproducir**
Verifica que el volumen del sistema esté activado, confirma que no hay otras aplicaciones bloqueando el audio, reinicia el secuenciador si persiste el problema y verifica la conexión de auriculares o altavoces.

**Audio entrecortado o con ruido**
Cierra otras aplicaciones que consuman CPU intensivamente, verifica que el sistema cumpla los requisitos mínimos, ajusta la configuración de efectos (reduce reverb/delay si es necesario) y considera usar una interfaz de audio dedicada.

**Latencia alta en modo manual**
El software está optimizado para latencia mínima pero factores del sistema pueden afectar la respuesta. Cierra aplicaciones innecesarias y considera actualizar drivers de audio.

### **Problemas de Interfaz**

**Ventana no responde**
El software incluye procesamiento multithread pero secuencias muy largas (>32 notas) con múltiples efectos pueden causar retrasos temporales. Usa el botón "Detener" si la aplicación parece congelada.

**Controles no funcionan**
Reinicia la aplicación completamente, verifica que no hay diálogos de error ocultos y usa el botón "Reset" para restaurar configuración por defecto.

### **Problemas de Rendimiento**

**Uso alto de CPU**
Desactiva efectos no esenciales (especialmente reverb y delay), reduce el número de notas en secuencias largas, cierra otras aplicaciones de audio simultáneas y verifica que el sistema cumpla requisitos recomendados.

**Archivo ejecutable no inicia**
Verifica que tienes permisos de ejecución en la carpeta, confirma que no hay software antivirus bloqueando la ejecución, extrae nuevamente el archivo ZIP y ejecuta como administrador si es necesario.

## **Consejos de Uso Profesional**

### **Flujo de Trabajo Creativo**

**Exploración Musical Sistemática**
Comienza con escalas familiares (Mayor/Menor) antes de explorar modos exóticos, experimenta con una escala a la vez para comprender su carácter único, usa el modo aleatorio para inspiración inicial y refina manualmente las ideas generadas.

Combina diferentes tipos de síntesis con efectos específicos, usa FM con delay para texturas complejas, experimenta con filtros en tiempo real y guarda configuraciones exitosas para proyectos futuros.
